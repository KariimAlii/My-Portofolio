                                                                              Patrick Kunka Geoffrey Weg • 5 years ago

                                                                              Hi Geoffrey, when we're searching the DOM using a text input to query elements, we are limited to what we can do with query selector strings unfortunately. This means it's not so good for any kind of intelligent searching with fuzzy logic and so on.

                                                                              The best way to achieve better searching, would be to implement a server-side search API that you can send requests do, and it returns, (for example) a list of IDs representing the matching items. You can then query the DOM for those IDs items using MixItUp's filter API:

                                                                              mixer.filter('#1, #23, #843, #22');

                                                                                    Patrick Kunka Geoffrey Weg • 5 years ago • edited

                                                                                    Sure. If you're using this demo as a boilerplate:

                                                                                    https://demos.kunkalabs.com...

                                                                                    You would just plug in a call to your analytics service in the filterByString() function, and you'll have access to the searchValue argument that's passed through.

                                                                                    However, if you're using the MultiFilter extension which has searching by text input built into it, it's going to be a little harder to hook in to, but still possible.

                                                                                    I would suggest adding the following code to callbacks.onMixStart callback. Note that data-filter-group attribute has been given a named value in the html ("search"), so we can retrieve it in our code.

                                                                                    HTML

                                                                                    ‹fieldset data-filter-group="search"›
                                                                                        ‹input type="text" data-search-attribute="class" placeholder="Search"/›
                                                                                    ‹/fieldset›

                                                                                    JavaScript

                                                                                    var mixer = mixitup(container, {
                                                                                        ...
                                                                                        callbacks: {
                                                                                            onMixStart: function() {
                                                                                                var searchValues = mixer.getFilterGroupSelectors('search');
                                                                                                var searchQuery = '';

                                                                                                if (searchValues.length > 1) {
                                                                                                    searchQuery = searchValues[0];

                                                                                                    // ... send `searchQuery` value to analytics service
                                                                                                }
                                                                                            }
                                                                                        }
                                                                                    });
