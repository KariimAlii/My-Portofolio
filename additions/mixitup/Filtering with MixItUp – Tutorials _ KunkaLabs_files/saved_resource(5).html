                                                                                              Patrick Kunka Karl 4 years ago

                                                                                              It sounds like you're almost there but the click handler is being invoked before the built-in one, and the built-in one gets pushed into the queue when it would normally just be aborted. Can you try this:

                                                                                              // Add custom click handler to each button
                                                                                              $('.control').on('click', function() {

                                                                                                  // Check if the clicked button is active
                                                                                                  if ( $(this).hasClass('mixitup-control-active') ) {
                                                                                                      // Clear all filters
                                                                                                      setTimeout(function() { mixer.filter('all'); }, 0);
                                                                                                  }
                                                                                              });

                                                                                              This will make sure your filter call is defered to the next tick, and the built-in handler will firstly be aborted as expected.
